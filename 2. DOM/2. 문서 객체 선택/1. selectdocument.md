# 문서 객체 가져오기

## 문서 객체 가져오기

## &nbsp;: 웹페이지(= 문서객체)에 이미 존재하는 HTML태그를 자바스크립트로 가져오는 방법

<br/>  
 
## 문서 객체 가져오기 첫번째 방식

## &nbsp;: getElementById(HTML id속성을 가져온다.)속성 사용

## &nbsp; → HTML id속성을 가져오는 방법을 일반적으로 많이 사용한다.

```html
<body>
  <!-- body태그 구성-->

  <h1 id="header-1">Header1</h1>
  <h1 id="header-2">Header2</h1>
</body>
```

<br/>

```javascript
// 문서 객체 가져오기
let header1 = document.getElementById('header-1');
let header2 = document.getElementById('header-2');

// 문서 객체의 innerHTML속성으로 태그의 텍스트 내용을 변경

header2.innerHTML = 'header2';
```
