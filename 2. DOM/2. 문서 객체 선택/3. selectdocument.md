# 문서 객체 가져오기

## 문서 객체 가져오기 세번째 방식

<br/>

## \- document.querySelector(선택자)

### &nbsp;: 선택자로 가장 처음 선택되는 문서 객체를 가져옴

<br/>

## \- document.querySelectorAll(선택자)

### &nbsp;: 선택자를 통해 선택되는 문서 객체를 배열로 가져옴

```html
<body>
  <!-- body태그 구성-->

  <h1 id="header-1">Header</h1>
  <h1 id="header-2">Header</h1>
</body>
```

<br/>

```javascript
// 문서 객체를 가져오기

// 아이디 선택자를 선택
let header1 = document.querySelector('#header-1');
let header2 = document.querySelector('#header-2');

// 문서 객체의 속성 변경
header1.innerHTML = 'header1'; // Header -> Header1
header2.innerHTML = 'header2'; // Header -> Header2
```
