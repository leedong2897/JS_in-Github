# 문서 객체 제거

## 문서 객체 제거

## &nbsp; → 문서 객체를 제거할 때는 removeChild(child)를 사용한다.

<br/>

## &nbsp; removeChild(child)

## &nbsp;&nbsp;&nbsp; : 문서 객체의 자식 노드를 제거한다.

### &nbsp;&nbsp; → 부모요소.removeChild(지울자식요소)

<br/>

```html
<body>
  <h1>Hello World</h1>
  <h2>Hello JavaScript</h2>
  <h3>나만 남았다.</h3>
</body>
```

<br/>

```javascript
let header1 = document.querySelector('h1');
let header2 = document.querySelector('h2');

//문서 객체 제거
document.body.removeChild(header1); //일반적
header2.parentNode.removeChild(header2);
```
